# 🎓 The Foundation School Website

A responsive and user-friendly school website built using **JSP, HTML, CSS, BOOTSTRAP**, and structured components. This project represents a complete front-end design for a school, showcasing programs, activities, and essential information.

## 📌 Features

* 🏫 Home page with overview of the school
* 📖 About page describing the institution
* 🎯 Programs section (Playgroup, Nursery, Kindergarten)
* 🎨 Activities page
* 📞 Contact page
* 🔁 Reusable components (Navbar & Footer using JSP)
* 💻 Responsive design with clean UI


## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, BootStrap
* **Backend (View Layer):** JSP (Java Server Pages)
* **Styling:** Custom CSS
* **Structure:** Component-based (Navbar & Footer)

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/TilakGollapudi/The-Foundation-School.git
```

### 2. Open in IDE

* Use **Eclipse / IntelliJ IDEA / VS Code**
* Make sure you have a **Java server (Apache Tomcat)** configured

### 3. Run the Project

* Deploy the project on **Tomcat Server**
* Open browser and go to:

```
http://localhost:8080/your-project-name/pages/home.jsp
```
